# ETboard - USB 데이터 통신
- PC와 ETboard를 USB로 데이터 통신하기  
- PC에서 파이썬 프로그램으로 ETboard의 LED ON/OFF 제어하기  
- 파이썬 & 마이크로파이썬   
#esp32 #uart #python #micropython #usb #pyserial

1. ETboard  
-. MicroPython v1.16

2. PC:  
-. Python v3.10  
-. pyCharm 또는 Jupyter Notebook  
-. 중요 모듈: pySerial
